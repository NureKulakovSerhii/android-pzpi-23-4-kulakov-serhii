ХІД РОБОТИ
Встановлення Android Studio
　　　　З офіційного сайту Android Studio було завантажено Android Studio останньої версії для операційної системи Windows. Під час встановлення було обрано компоненти Androis SDK та AVD Manager для встановлення. Для перевірки того, що все працює належним чином було створено новий проєкт. Після цього до проєкта був підключения емулятор.
Створення мінімального життєвого Android-застосунку.
　　　　Після запуску та налаштування Android Studio було створено новий проєкт на мові програмування Java. Після створення проєкту та підключення емулятору мною було вивчено головні файли проєкту: MainActivity.java, activity_main.xml та  AndroidManifest.xml. Оскільки емулятор було підключено в попередньому пункті, було натиснуто кнопку “Run” для перевірки успішної компіляції застосунку. Далі у файл activity_main.xml була додана кнопка:
　　　　
1.<Button
2.    android:id="@+id/myButton"
3.    android:layout_width="wrap_content"
4.    android:layout_height="wrap_content"
5.    android:text="Натисніть мене" />

У файл MainActivity.Java була додана обробка натискання на кнопку:
1.Button myButton = findViewById(R.id.myButton);
2. TextView myTextView = findViewById(R.id.myTextView);
 3.myButton.setOnClickListener(view -> {
 4.   myTextView.setText("Текст змінено!");
 5. });

В результаті натискання на кнопку текст змінювався.
	1.2.3 Публікація коду на GitHub.
	Налаштував Git в Android Studio, перевірив шлях до виконуваного файлу Git, ініціалізував систему контролю версій для проєкту. Створив новий репозиторій на GitHub, підключив локальний репозиторій до GitHub і опублікував проєкт.
